# Jogo do Pong

Jogo de Pong em 3D utilizando Iluminação.

> _**Como Jogar**_: 

* Para mover a palheta próxima ao monitor apertar as setas da esquerda e da direita.
* Para mover a palheta ao fundo, apertar as teclas A e D _(Maiúsculas ou Minúsculas)_.

## Motivação

Este jogo foi feito como Projeto Final na disciplina de **Computação Gráfica** do curso de **Bacharelado em Ciência da Computação** cursado no **CEFET/RJ**.

## Linguagem
Este jogo foi programado em Python *3.6.0* utilizando OpenGL *3.1.0*.
